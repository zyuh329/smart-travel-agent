# 智游AI - 智能旅游规划平台

基于 Multi-Agent 架构的智能旅游网站前端原型

## 项目结构

```
smart-travel-agent/
├── index.html    # 主页面
├── styles.css    # 样式文件
├── app.js        # 交互逻辑
└── README.md     # 说明文档
```

## 功能演示

1. **多 Agent 协作流程展示**
   - 需求解析 Agent → 规划 Agent → 资源匹配 Agent → 优化 Agent
   - 动画展示每个 Agent 的处理状态

2. **长链推理模拟**
   - 模拟 4 个阶段智能处理流程
   - 展示从需求理解到行程生成的全过程

3. **行程结果展示**
   - 5天4晚云南昆大丽行程
   - 详细时间线、景点、交通、餐饮推荐
   - 费用预算明细

## 使用方式

直接在浏览器中打开 `index.html` 文件即可运行。

## 技术亮点

- 多 Agent 协作架构可视化
- 响应式设计，支持移动端
- CSS 动画增强用户体验
- 模拟 AI 推理过程

## 扩展方向

后端可接入真实大模型 API（如 GPT-4、Claude、文心一言）实现真正的智能规划功能。
